## Installation

Install via npm:
```bash
npm install squeakjs-embed-kit
```

## Usage

Copy or reference the files in `node_modules/squeakjs-embed-kit/`:
```html
<!-- Adapt the path below if needed -->
<script src="node_modules/squeakjs-embed-kit/squeakjs/squeak.js"></script>
<link rel="stylesheet" href="node_modules/squeakjs-embed-kit/squeakjs/run/squeakjs.css">
<!-- And so on -->
```
Or start with `index.html` as a base.
